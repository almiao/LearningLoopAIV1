# Generated Review Output

This directory is reserved for runtime-generated session review artifacts.

## What belongs here

- `index.json`
- `index.md`
- per-scenario `*.json`
- per-scenario `*.md`

## Rules

- Treat this folder as disposable output.
- Do not use it as long-term documentation storage.
- Historical snapshots belong under `archive/session-review-snapshots/`.
